WELCOME TO CS181 at U of M

This is a demo page for the CS 181 HTML Project. This project is designed to help you understand the basics of HTML coding. However, don't expect to be a web-design guru by the end of this project. HTML requires a lot of practice and there is always more to learn. This class will really help you get started though, and you should be able to learn new skills very easily. We have included below some additional links which can help you learn HTML and other web-technologies.

Given below is a list of things you need to do for the first part of this project:

  • A title in the title bar
  • The body of the document should have a background. This may be a color, a public domain image or an image for which you own the copyright.
  • Use heading text at the top of the document body (use the <H#> tag)
  • Add at least four sentences of text
  • Include a list with at least 3 items (like this one!). To create the list, you must use the list tags.
  • Add a short summary of your powerpoint presentation and make a link to the PDF version of it.
  • Make four links to other webpages:
    1. One link to the class homepage (http://www.umich.edu/~cs181)
    2. One link to the persuasion page you will make later in the project.
    3. Two different links to sites that contain information you are interested in.
  • Add two pictures to the page.
